### コンポーネント化(3章)
- 再利用性が高まり、開発効率をあげられる
- 既に使用されているコンポーネントを再利用することで、品質を保てる
- コンポーネントを適切に区切り、疎結合にした場合、保守性が高まる
- カプセル化されて開発で意識すべき範囲を限定できるようになる
